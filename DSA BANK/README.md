Bank works in two modes:

1. [BANKING Mode](#banking-mode)
2. [ATM Mode](#ATM-mode)

to switch between the modes use command Z

# BANKING Mode

| Command | Description                    |
| ------- | ------------------------------ |
| S       | open bank                      |
| O       | open an account                |
| B       | balance enquiry                |
| W       | withdrawal                     |
| D       | deposit                        |
| C       | close account                  |
| A       | activate ATM                   |
| U       | unlock ATM                     |
| P       | print all accounts             |
| I       | apply interest to all accounts |
| E       | exit and close all accounts    |

# ATM Mode

| Command | Description     |
| ------- | --------------- |
| X       | PIN change      |
| F       | fund transfer   |
| B       | balance enquiry |
| W       | withdrawal      |
| M       | mini statement  |
